# `@neuton/saas-ui`

> saas 组件库

## 安装

### 添加私有 npm

1. 在项目根目录下创建.npmrc 文件。
2. 在.npmrc 文件中写入如下内容:

```
registry=http://82.157.120.5:4873/
```

### 安装@neuton/saas-ui

```
# 使用yarn
yarn add @neuton/saas-ui
# 使用npm
npm i @neuton/saas-ui
# 使用pnpm
pnpm i @neuton/saas-ui
```
